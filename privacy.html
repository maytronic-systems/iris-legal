2. General Information

The app IRIS – Insight & Recon Interface System provides analysis and visualization of system-related information on Android devices, including app activity, permissions, and network statistics.

All data processing is performed exclusively locally on the user’s device. No personal data is transmitted to external servers.

IRIS does not create user profiles, does not track users, and does not collect personal content, location data, or communication data.

3. Types of Data Processed

Depending on the enabled features, IRIS may process the following data locally on the device:

  • List of installed applications
  • Application names, package names, and icons
  • Granted application permissions
  • Application usage statistics (foreground and background activity)
  • Aggregated network traffic data (upload and download volumes)
  • User-defined rules, alerts, and configuration settings

All data remains stored exclusively on the user’s device.

4. Android Permissions

a) Usage Access (PACKAGE_USAGE_STATS)

Required to display application usage duration, foreground/background activity, and activity timelines.

b) Network Statistics Access

Required to display application-level network traffic and aggregated data usage.

c) Installed Applications Access

Required to identify installed applications and analyze permissions and metadata.

d) Notifications (optional)

Used exclusively for event-based notifications triggered by user-defined rules (e.g. configured alerts in the Action Center).

5. Purpose of Data Processing

Data processing is carried out solely for the following purposes:

  • Analysis and visualization of application activity
  • Display of system and usage statistics
  • Transparency regarding application permissions
  • Event-based notifications according to user configuration

6. Legal Basis

Data processing is based on the following legal grounds:

  • Article 6(1)(b) GDPR – performance of the app’s functionality
  • Article 6(1)(a) GDPR – user consent by granting system permissions

7. Data Storage and Deletion

All data is stored locally on the user’s device. No cloud synchronization or external storage is used.

All stored data is deleted when the app is uninstalled.

8. Data Sharing

No personal data is shared with third parties.

9. Third-Party Services

IRIS does not use external analytics, tracking, or advertising services.

If in-app purchases are offered, payment processing is handled exclusively by the Google Play Store in accordance with Google’s privacy policies.

10. User Rights

Users have the right to:

  • Access their data
  • Request correction or deletion
  • Restrict processing
  • Withdraw consent at any time

Since all data is stored locally, these rights can be exercised directly through the app or by uninstalling it.

11. Supervisory Authority

Users may lodge a complaint with a competent data protection supervisory authority.

This privacy policy may be updated to reflect functional or legal changes.
